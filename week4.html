WELCOME TO WEEK 4

In Week 4, we focused on using functions in p5.js to create illustrations.
By building characters through modular code, we practiced drawing with geometric shapes such as ellipses, rectangles, and arcs.
The goal was to combine these shapes into faces, animals, and abstract compositions.

drawing with p5js


function drawPortrait(x, y) {
  push();
  translate(x, y);
  scale(1.8);
  noStroke();
  fill('#B48A78');
  rect(-80, -10, 160, 100);
  fill('#FFE600');
  ellipse(0, 0, 160, 160);
  fill('#FFE600');
  ellipse(-85, 0, 30, 30);
  ellipse(85, 0, 30, 30);
  fill('#B48A78');
  beginShape();
  vertex(0, -70);
  bezierVertex(-60, -60, -60, 20, 0, 40);
  bezierVertex(-20, 20, -20, -60, 0, -70);
  endShape(CLOSE);
  beginShape();
  vertex(0, -70);
  bezierVertex(60, -60, 60, 20, 0, 40);
  bezierVertex(20, 20, 20, -60, 0, -70);
  endShape(CLOSE);
  drawCuteDotEye(-50, 10);
  drawCuteDotEye(50, 10);
  stroke('#111');
  strokeWeight(3);
  noFill();
  arc(-35, -35, 5, 5, PI, TWO_PI);
  arc(35, -35, 5, 5, PI, TWO_PI);
  strokeWeight(4);
  beginShape();
  curveVertex(-10, 20);
  curveVertex(-5, 24);
  curveVertex(0, 22);
  curveVertex(5, 24);
  curveVertex(10, 20);
  endShape();
  pop();
}
